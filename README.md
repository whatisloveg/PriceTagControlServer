Для раввертывания на сервере:
1) sudo apt update
2) sudo apt install docker.io
3) sudo apt install docker-compose
4) загрузить проект с помощью filzilla
5) перехоидм с помощью cd в деректорию с проектом
6) docker-compose up -d
